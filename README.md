# 2048
A simple 20248 game using a 2d matrix, functional programming, VueJS3, Typescript and ViteJS.

## Prequisites
- npm >= v10.*
- nodejs >= v20.*

## Installation
```bash
npm install
```
Install Husky locally:
```bash
npm run prepare
```

## Usage
Run a dev environment:
```bash
npm run dev
```
Run linter and prettier:
```bash
npm run lint:fix
```
